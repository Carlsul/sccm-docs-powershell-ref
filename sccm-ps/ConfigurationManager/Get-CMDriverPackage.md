# Get-CMDriverPackage

## SYNOPSIS
Gets a driver package.

## SYNTAX

### SearchByName (Default)
```
Get-CMDriverPackage [-Fast] [-Name <String>] [-DisableWildcardHandling] [-ForceWildcardHandling]
 [<CommonParameters>]
```

### SearchByIdMandatory
```
Get-CMDriverPackage [-Fast] -Id <String[]> [-DisableWildcardHandling] [-ForceWildcardHandling]
 [<CommonParameters>]
```

## DESCRIPTION
The **Get-CMDriverPackage** cmdlet gets a driver package.

> [!NOTE]
> Configuration Manager cmdlets must be run from the Configuration Manager site drive.
> The examples in this article use the site name **XYZ**. For more information, see the
> [getting started](/powershell/sccm/overview) documentation.

## EXAMPLES

### Example 1: Get a driver package that is specified by its identifier
```
PS XYZ:\> Get-CMDriverPackage -Id "CM100042"
```

This command gets a driver package that is specified by its identifier.
